# 공동 시간외근무표 MVP

## 실행
1. Node.js 20 이상 설치
2. 이 폴더에서 `npm install`
3. `npm start`
4. 브라우저에서 `http://localhost:3000`

같은 서버 주소에 접속한 모든 사용자가 같은 표를 실시간으로 수정합니다.

## 포함 기능
- 로그인 없는 공동 접속
- 행 추가/수정/삭제
- Socket.IO 실시간 반영
- JSON 파일 영구 저장
- 근무시간 자동 계산
- Excel 다운로드
- A4 가로 인쇄
- 모바일 대응

## 온라인 배포
Node.js 장기 실행과 쓰기 가능한 디스크를 제공하는 서버에 배포하세요. 데이터는 `data/rows.json`에 저장됩니다. 여러 서버 인스턴스로 확장하려면 PostgreSQL과 Socket.IO Redis 어댑터로 교체해야 합니다.

## 주의
로그인이 없으므로 URL을 아는 사람은 누구나 수정할 수 있습니다. 실제 개인정보를 넣는 공개 인터넷 서비스에는 그대로 사용하지 마세요.
