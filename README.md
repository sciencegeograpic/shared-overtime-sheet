# 공동 업무표 v6 - 주간계획 서식 수정

## 수정 사항

- 주간계획 PDF를 더 이상 한 페이지에 두 장 모아찍기하지 않습니다.
- 원본처럼 A4 세로 기준으로 `노년사회화팀·지역복지팀` 1쪽, `총무팀·영양팀·아중노인복지관` 1쪽으로 각각 출력합니다.
- 주간계획 제목, 결재칸, 팀명/기간 라인, 표 컬럼 비율, 세로 구분 칸을 원본에 더 가깝게 조정했습니다.
- 시간외근무표, 차량일지, 신용·체크카드 사용내역 기능은 유지합니다.
- 외부 npm 패키지 없이 Node.js 기본 기능만 사용합니다.

## 적용

기존 GitHub 폴더에 전체 덮어쓰기 후 GitHub Desktop에서 `Replace with v6 weekly format` 커밋 및 Push 하세요.
